Pour le serveur :

1. Placer le dossier api, dans le répertoire www de votre serveur web.
2. Pour lancer le websocket pour le chat :
	- se positionner dans le répertoire bin du dossier "ws"
	- ensuite lancer un terminal depuis ce dossier
	- exécuter la commande : php Server.php
3. Récupérer votre adresse ip
4. Remplacer dans le fichier de ressource "strings" dans le champs "url" le mot IP par l'adresse de votre serveur
5. Lancer l'application
